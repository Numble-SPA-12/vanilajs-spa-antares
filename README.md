## How to start

### Prerequisites

- 최신 버전의 `yarn`이 설치되어 있어야 합니다.
- 전역에 [`serve`](https://github.com/vercel/serve) 패키지가 설치되어 있어야 합니다.

### 개발환경 실행

```bash
yarn dev
```

## 라이브러리

- 패키지 매니저: [yarn v3](https://yarnpkg.com/)
- CSS 프레임워크: [tailwindcss](https://tailwindcss.com/)
- http: [axios](https://axios-http.com/)
